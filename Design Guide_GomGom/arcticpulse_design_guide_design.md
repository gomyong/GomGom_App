# ArcticPulse Design System Guide

## 1. Overview
ArcticPulse는 북극곰의 경로를 추적하고 연구 데이터를 관리하는 전문가용 모니터링 시스템입니다. 미니멀한 라인 아트 스타일과 지형 지도 감성을 결합하여 데이터의 명확성과 예술적 가치를 동시에 제공합니다.

## 2. Visual Identity
- **Tone & Manner**: Minimalist, Professional, Scientific, Artistic Line Art
- **Key Visuals**: Isometric views, Grid systems, Bold stroke-based iconography

## 3. Colors
기존 디자인 시스템({{DATA:DESIGN_SYSTEM:DESIGN_SYSTEM_1}})의 색상 팔레트를 따릅니다.
- **Surface**: #F7F9FB (Clean, snowy background)
- **Primary**: #007AFF (Trustworthy, scientific blue)
- **Outline**: #D8DADC (Subtle grid and separation lines)
- **Warning**: High contrast colors for alerts (e.g., Red for proximity warnings)

## 4. Typography
- **Primary Font**: Hanken Grotesk
- **Styles**:
  - Headline: Bold, for screen titles and critical metrics.
  - Body: Regular/Medium, for detailed research data.
  - Label Caps: For navigation and status indicators.

## 5. Iconography
- **Style**: High-fidelity line art with consistent stroke weights.
- **Bear Markers**:
  - Profile View ({{DATA:IMAGE:IMAGE_11}}): Used for real-time map tracking to indicate movement.
  - Portrait View ({{DATA:IMAGE:IMAGE_12}}): Used in lists and profiles for individual identification.

## 6. Components
### Top Bar
- Leading: Biotech icon for research context.
- Title: ArcticPulse brand logo.
- Trailing: Settings for system configuration.

### Navigation (Bottom Bar)
- **Map**: Real-time telemetry view.
- **Bears**: Subject listing and status.
- **Alerts**: System notifications and field updates.

### Data Cards
- Minimalist borders with ample whitespace.
- Key metrics (Heart Rate, Temp) displayed with clear hierarchy.

## 7. Layout Principles
- **Grid**: 8dp spacing system for consistent alignment.
- **Whitespace**: Generous padding to reduce cognitive load during field research.
- **Map View**: Isometric perspective with 2D grid overlays for spatial orientation.
